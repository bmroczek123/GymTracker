Aplikacja webowa służąca do rejestrowania treningów oraz wizualizacji postępów siłowych. 

## Jak uruchomić aplikację?

Do poprawnego działania aplikacji wymagany jest zainstalowany **Docker Desktop**.

### 1. Pobierz repozytorium
```bash
git clone [https://github.com/bmroczek123/GymTracker.git](https://github.com/bmroczek123/GymTracker.git)
cd GymTracker

```

### 2. Uruchom kontenery

Użyj Docker Compose, aby automatycznie pobrać obrazy, zbudować aplikację i skonfigurować bazę danych:

```bash
docker-compose up --build

```

### 3. Dostęp do aplikacji

Aplikacja będzie dostępna pod adresem:
**http://localhost:3000**
